# SUBLIMINAL

> Passive-first subdomain reconnaissance tool  
> **Educational purposes only**

---

## Overview
SUBLIMINAL is an asynchronous OSINT-based subdomain enumeration tool with optional limited active enrichment using TLS certificate SAN extraction.

Designed to be fast, quiet, and suitable for bug bounty and penetration testing reconnaissance workflows.

---

## Features
- Passive subdomain enumeration
- Multiple OSINT sources
- Optional TLS SAN enrichment
- Async & high performance
- Clean output
- Ctrl+C safe termination
- Kali Linux compatible

---

## Requirements
- Python 3.9+
- httpx
